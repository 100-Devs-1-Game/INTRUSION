INTRUSION is a Fixed-Camera Psychological Horror / Systems Management game, inspired by the mechanics of Five Nights at Freddy's.

Contain the breach. Survive the shift. Never leave the chair.

INTRUSION is part of the 33 games made in 100 days gamejam game hosted by CGGJ (Collaborative Godot Game Jam)
